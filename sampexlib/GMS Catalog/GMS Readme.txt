Description: This is a Geomagnetic Storm (GMS) Catalog created to allow a better understanding of the behavior of 
microbursts recorded by the SAMPEX satellite during various types of storms in the 1990s. The main function of this
catalog is to provide an organized database of approximately weeklong storms.
